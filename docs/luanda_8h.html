◆ luanda_breakpoint_t

typedef int luanda_breakpoint_t

A handle to a breakpoint set in a specific indefior process under the controll of the Luanda debugger.

A 'luanda_breakpoint_t' handle is used to refer to a breakpoint in Luanda debugger calls.

A 'luanda_breakpoint_t' handle is only meaningful given a specific inferior. That is, the handle must always be used in conjuction with a 'luanda_inferior_t'.

◆ luanda_inferior_t

typedef int luanda_inferior_t

A handle to an inferior process under the controll of the Luanda debugger.

A 'luanda_inferior_t' is used to refer to an inferior when making Luanda API calls.
